# everymorning

A daily STEM paper digest service.

## Overview

everymorning is a service that delivers curated STEM research papers and summaries to users every morning.

## Project Structure

```
everymorning/
├── backend/          # Backend services and models
├── frontend/         # Frontend pages and components
├── templates/        # Email and other templates
├── tests/            # Test suite
├── docs/             # Documentation
├── .github/          # GitHub workflows and configuration
└── scripts/          # Utility scripts
```
